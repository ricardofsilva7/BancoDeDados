## 📌 Sobre

Repositório com exercícios e projetos da disciplina de **Banco de Dados** do curso de Análise e Desenvolvimento de Sistemas (SENAI).

Conteúdo abrange desde **modelagem de dados (ER)** até **manipulação avançada com SQL (DDL, DML, DQL)**.

## 🎯 Conceitos Abordados

- **Modelagem de Dados**: Diagramas Entidade-Relacionamento (ER)
- **DDL** (Data Definition Language): `CREATE`, `ALTER`, `DROP`
- **DML** (Data Manipulation Language): `INSERT`, `UPDATE`, `DELETE`
- **DQL** (Data Query Language): `SELECT`, `JOIN`, `GROUP BY`, subconsultas
- **Normalização**: 1FN, 2FN, 3FN
- **Projeto Integrado**: Modelo completo de banco para sistema universitário
